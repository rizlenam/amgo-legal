# Suppression des données - Am'Go

## Comment supprimer vos données

Am'Go respecte votre vie privée et vous permet de supprimer vos données à tout moment.

### Données collectées

Am'Go collecte uniquement :
- Votre position GPS (pour afficher les places de parking)
- Un identifiant anonyme de votre appareil
- Les places que vous signalez
- Votre score de points

### Suppression automatique

- **Places signalées** : Supprimées automatiquement après expiration (5-120 minutes selon la zone)
- **Position "MA VOITURE"** : Supprimable manuellement dans l'app (appui long sur le marqueur)

### Suppression complète de votre compte

Pour supprimer **toutes** vos données définitivement :

**Option 1 : Depuis l'application**
1. Désinstallez l'application Am'Go de votre appareil
2. Vos données seront automatiquement supprimées de nos serveurs dans les 30 jours

**Option 2 : Par email**
Envoyez un email à rizlen.am@gmail.com avec :
- Objet : "Suppression de mes données Am'Go"
- Dans le message : "Je souhaite supprimer toutes mes données personnelles de l'application Am'Go"

Nous traiterons votre demande sous **7 jours ouvrés** et vous enverrons une confirmation par email.

### Données anonymes

Certaines statistiques anonymisées (nombre total de places partagées globalement) peuvent être conservées à des fins d'amélioration du service, mais **aucune donnée personnelle identifiable** n'est conservée après suppression.

### Questions ?
Pour toute question concernant vos données personnelles, contactez-nous à rizlen.am@gmail.com
